# Java Number Programs

This project contains 14 important number-based programs implemented using Core Java.

## Programs Included
1. Factorial
2. Fibonacci
3. Prime Number
4. Armstrong Number
5. Strong Number
6. Happy Number
7. Magic Number
8. Perfect Number
9. Spy Number
10. Neon Number
11. Palindrome Number
12. Reverse Number
13. Xylem / Phloem
14. Disarium Number

## Concepts Used
- Loops
- Switch Case
- Methods
- Scanner Class
- Logic Building
